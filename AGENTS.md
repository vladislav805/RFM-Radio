# Repository Guidance for AI Agents

Project-specific technical context is stored in [`.agents/`](.agents/README.md).

Before changing the Qualcomm native FM implementation, read
[`.agents/native-fm-backend.md`](.agents/native-fm-backend.md). It is the
canonical architecture and behavior map for `native/` and its Android bridge.

The README files under `native/` contain useful background but have stale file
names and an obsolete event-protocol description. Prefer the `.agents`
documentation and verify behavior against source code.
